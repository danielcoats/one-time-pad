The One-Time Pad

The One-Time Pad is a method of encryption and the only known perfect cipher. If used correctly, a message encrypted using the One-Time Pad is impossible to crack.

One-Time Pad

Enter your secret message and a random key at least as long as the message. If you leave the Secret Key field empty, a key will be automatically generated for you. Click Encrypt and the ciphertext will be generated from the message and key using the algorithm described below. Paste the ciphertext back into the message box, enter your secret key, and click Decrypt to see the ciphertext decoded.

Rules

In order for your encrypted message to be unbreakable, you must follow these rules:

  • The key must be at least as long as the message that you want to send.
  • Each key must be used only once, and must be destroyed after use.
  • Only the sender and receiver may have a copy of the key.
  • The key must be random i.e. not generated by a computer algorithm.

How it works

Enciphering

  1. Assign a number to each letter of the alphabet. For example, A = 0, B = 1 … Z = 25.
  2. Take your secret message. For example, SECRETCODE.
  3. Turn your secret message into digits using the mapping above. For example, 18 04 02 17 04 19 02 14 03 04.
  4. Create your random key, which is at least as long as the message, for example EQUHESEVZO, and convert it into digits 04 16 20 07 04 18 04 21 25 14.
  5. Now add each letter in the message to the corresponding digit in the key using modular arithmetic. For example, S + E mod 26 = 18 + 4 mod 26 = 22 = W and O + V mod 26 = 14 + 21 mod 26 = 35 mod 26 = 9 = J. Enciphering SECRETCODE with the key EQUHESEVZO gives WUWYILGJCSâ€”the ciphertext.
  6. Send the ciphertext, and the ciphertext only, to your friend, who must already know the secret key.

Deciphering

  1. Take the ciphertext WUWYILGJCS that you have received and convert it into digits as above, which gives 22 20 22 24 08 11 06 09 02 18.
  2. Now subtract each letter in the secret key from the ciphertext. For example, W - E mod 26 = 22 - 4 mod 26 = 18 = S. Repeat this process for each letter in the ciphertext.
  3. And there you have it, the secret message: SECRETCODE. Now make sure to destroy your secret key.
