One-Time Pad An online demonstration

The One-Time Pad is a method of encryption and the only known perfect cipher. If used correctly, a message encrypted using the One-Time Pad is impossible to crack.

One-Time Pad

Rules

  • The key must be at least as long as the message that you want to send.
  • Each key must be used only once, and must be destroyed after use.
  • Only the sender and receiver may have a copy of the key.
  • The key must be truly random.

How it works

Enciphering

  1. Take a secret message. For example, SECRETCODE.
  2. Assign a number to each letter of the alphabet. For example, A = 0, B = 1 … Z = 25.
  3. Turn your secret message into ASCII digits, For example, 83 69 67 82 69 84 67 79 68 69.
  4. Create your random key, which is at least as long as your message, for example, EQUHESEVZO, and convert it to ASCII, 69 81 85 72 69 83 69 86 90 79.
  5. Now add each ASCII code in your secret message to the corresponding ASCII code in your key and subtract 64. If the result is greater than 90, then subtract 26. For example, S + E = 83 + 69 - 64 = 88 = X. Enciphering SECRETCODE with the key EQUHESEVZO gives XVXZJMHKDT, the cipher.
  6. Send the cipher, and the cipher only, to your friend, who must already know the secret key.
